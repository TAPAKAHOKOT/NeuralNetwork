# NeuralNetwork

After run, you can load previously trained weights or start the workout again.
You can abort your workout by pressing enter

After of loading or training weights you can ask the neural network "what is this figure"
To do this, hold down the left mouse button to draw 1 or 0
And press the right mouse button to find out the output of the neural network

To add your training example, you can draw 1 or zero scroll up (to add 1) or down (for 0).
